# Projeto Django de Estudos

![Django Logo](https://www.djangoproject.com/m/img/logos/django-logo-negative.png)

Um projeto Django simples para ajudar você a começar sua jornada de aprendizado em Django. Este projeto utiliza Django 4.2.5 e outras bibliotecas relacionadas.

## Requisitos

Certifique-se de que você tenha as seguintes bibliotecas instaladas:

- asgiref==3.7.2
- Django==4.2.5
- python-dotenv==1.0.0
- sqlparse==0.4.4
- typing_extensions==4.8.0
